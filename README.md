Sprout.io 🌱 | Smart Irrigation System

Sprout is an innovative agricultural solution designed to automate plant care using Vision AI and LEGO Spike Prime robotics. Built by Group A (Year 7 Adeshina students) at Greensprings School, Anthony Campus.

🚀 Features

Vision AI Analysis: Uses camera input to determine plant health and hydration needs.

LEGO Spike Prime Integration: Seamless wireless communication to trigger physical irrigation hardware.

PWA Ready: Installable on Android and iOS devices directly from the browser.

Modern UI: Clean, responsive interface built with Tailwind CSS.

🛠️ Tech Stack by FolatheDuckofDuckingburg 

Frontend: HTML5, Tailwind CSS, Lucide Icons

Logic: JavaScript (ES6+)

Robotics: LEGO Spike Prime Hub & Python API

AI: Computer Vision Model (Integrated via Web API)
